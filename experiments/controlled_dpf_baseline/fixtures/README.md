# Controlled Baseline Fixtures

Current MP5-MP8 scope is limited to the two clean-room range-bearing fixtures:

- `range_bearing_gaussian_moderate`;
- `range_bearing_gaussian_low_noise`.

The fixture contract is defined in
`docs/plans/bayesfilter-student-dpf-baseline-clean-room-controlled-baseline-spec-2026-05-13.md`.

Do not add AR(2), structural-SSM, stiffness, resampling-stress, or other broad
fixture families under this lane unless a later student-baseline master revision
explicitly authorizes them.
