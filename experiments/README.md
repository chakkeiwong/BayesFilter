# BayesFilter Experiments

This directory contains quarantined experimental projects.  Code here supports
testing, comparison, debugging, and report generation.  It is not production
BayesFilter code.

Hard rule:

```text
No module under bayesfilter/ may import from experiments/.
```

Experimental results may inform benchmarks, plans, and implementation choices,
but they do not certify correctness, HMC readiness, or production release
status.
